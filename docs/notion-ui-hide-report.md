# Notion UI Hide Report

작성일: 2026-03-24

## 목적

- 현재 브랜치에서는 Notion 기능 자체를 제거하지 않고, 설정 화면에서만 노출을 숨긴다.
- 출시 문서와 사업 문서는 `출시될 제품 상태`를 기준으로 유지한다.

## 이번 조치

- 옵션 화면의 `Notion` 저장 대상 선택지를 숨김
- 옵션 화면의 `Notion 연결` 섹션 숨김
- 옵션 화면의 Notion 비교표 행 숨김
- 옵션 화면 상단 카피를 현재 노출 기준에 맞게 Obsidian 중심으로 조정

## 유지한 범위

- Notion 백엔드 코드
- Notion OAuth/저장 관련 타입과 로직
- 출시 기준 README 및 사업계획서의 Notion 설명

## 의도

- 외부 검토나 운영 준비가 끝나기 전까지는 일반 사용자 UI에서 Notion을 노출하지 않는다.
- 다만 문서 기준은 임시 브랜치 상태가 아니라 `출시 목표 상태`에 맞춘다.
