ZIP 다운로드 제품 페이지
ZIP 배포 개발자 모드 필요 자동 업데이트 없음

현재는 Chrome Web Store 배포가 아니라 ZIP 설치입니다. 한 번만 등록하면 바로 사용할 수 있습니다.

왜 개발자 모드가 필요한가요?

아직 ZIP 배포 단계라 Chrome의 압축해제 확장 로드 기능을 사용합니다. 설치 후 일반 사용 자체가 위험해지는 것은 아닙니다.

PC 설치 안내

Chrome 개발자 모드로 extension 설치

현재는 ZIP 배포입니다. 아래 순서대로 한 번만 등록하면 됩니다.

  1. 1
    ZIP을 내려받고 압축을 풉니다

    ss-threads-extension.zip을 먼저 압축 해제하세요. ZIP 파일 자체는 바로 선택하면 안 됩니다.

    ZIP 다운로드
  2. 2
    Chrome에서 extensions 페이지를 엽니다

    주소창에 chrome://extensions를 입력해 이동합니다.

  3. 3
    우측 상단 개발자 모드를 켭니다

    토글을 켜면 압축해제된 확장 프로그램을 로드합니다 버튼이 나타납니다.

  4. 4
    압축해제된 확장 프로그램을 로드합니다를 클릭합니다

    파일 선택 창이 열리면, 방금 압축 해제한 폴더를 지정할 준비를 합니다.

  5. 5
    폴더를 선택해 설치를 완료합니다

    압축을 푼 뒤 생긴 ss-threads-extension 폴더를 선택하면 설치가 완료됩니다.

  6. 6
    Threads에서 바로 확인합니다

    설치 후 확장 아이콘을 고정하고 Threads 개별 포스트 페이지에서 저장을 한 번 테스트하면 됩니다.

ZIP 설치 메모

자동 업데이트가 아직 없으므로 새 버전이 나오면 ZIP을 다시 받아 같은 방식으로 교체 설치하면 됩니다.

설치 확인

    __SITE_HOST__
